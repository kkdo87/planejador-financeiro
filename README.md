# PoupApp - Seu Assistente de Economia

**PoupApp** é seu assistente financeiro pessoal, projetado para ajudar você a gerenciar seus gastos, economizar dinheiro e conquistar seus objetivos financeiros.

## 🚀 Funcionalidades

- 📊 **Dashboard Inteligente**: Visualize suas finanças de forma clara e intuitiva
- 💰 **Gestão de Gastos**: Controle gastos fixos e variáveis com sugestões de economia
- 📈 **Relatórios Detalhados**: Análises completas da sua situação financeira
- 🤖 **Diagnóstico com IA**: Receba um plano personalizado para sair das dívidas
- 🔔 **Notificações**: Alertas inteligentes sobre prazos e limites
- 💬 **Suporte 24/7**: Chat de suporte sempre disponível
- 📱 **Design Responsivo**: Funciona perfeitamente em qualquer dispositivo

## 🛠️ Tecnologias

Este projeto foi construído com:

- [Next.js 15](https://nextjs.org) - Framework React
- [TypeScript](https://www.typescriptlang.org/) - Tipagem estática
- [Tailwind CSS](https://tailwindcss.com/) - Estilização moderna
- [Supabase](https://supabase.com/) - Backend e autenticação
- [OpenAI](https://openai.com/) - Inteligência artificial
- [Recharts](https://recharts.org/) - Gráficos interativos

## 🚀 Como Começar

Primeiro, instale as dependências:

```bash
npm install
```

Em seguida, execute o servidor de desenvolvimento:

```bash
npm run dev
```

Abra [http://localhost:3000](http://localhost:3000) no seu navegador para ver o resultado.

## 📝 Configuração

1. Configure suas variáveis de ambiente no arquivo `.env.local`
2. Configure sua conta Supabase
3. Adicione sua chave da API OpenAI

## 🌟 Sobre o PoupApp

O PoupApp foi criado para democratizar o acesso a planejamento financeiro inteligente, usando o poder da inteligência artificial para ajudar pessoas a tomarem melhores decisões financeiras e alcançarem a liberdade financeira.
